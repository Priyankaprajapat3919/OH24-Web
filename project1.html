Employee Management system

Description: Employee data need to be maintained in any company. Each company has an employee with a unique employee id, employee role, etc. All of this data is maintained in a system employee management system, where all the data about each employee is stored we can fetch, update and add data to this system. Using C we can create an employee management system that can perform all these tasks, using basic C knowledge like string, array, etc. The functionality of the Employee Management System is mentioned below:
1. Built The Employee Table.
2. Insert New Entries.
3. Delete The Entries.
4. Search A Record.
